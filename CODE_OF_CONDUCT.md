# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind.
- Don't harass anyone for any reason.
- If someone is being a problem, report it to the maintainer.

## Reporting

Email the maintainer: see the repository owner's contact on GitHub.

Confidentiality respected. Bad-faith reports will be ignored. Good-faith reports get a response within a few days.

## Enforcement

Three-strike pattern:

1. Private warning.
2. Temporary ban from the issue tracker / discussion (typically 30 days).
3. Permanent ban.

Severe violations (harassment, threats, doxxing) skip straight to permanent ban.
